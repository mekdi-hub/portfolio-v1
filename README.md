# Portfolio Website with Admin Panel

A modern, full-stack portfolio website built with React (frontend) and Laravel (backend), featuring a complete admin panel for content management.

## Features

### Frontend
- 🎨 Modern, animated UI with custom cursor
- 🌟 Particle and star background effects
- 📱 Fully responsive design
- 🎯 Smooth section navigation
- 📊 Skills showcase with animations
- 💼 Projects gallery
- 📧 Contact form


### Backend
- 🚀 Laravel REST API
- 💾 My sql
- 🔌 RESTful endpoints
- ✅ Input validation
- 🌐 CORS enabled

## Tech Stack

### Frontend
- React 19
- Vite
- Three.js (for 3D effects)
- Lenis (smooth scrolling)
- CSS3 animations

### Backend
- Laravel 11
- PHP 8.2+
 -My sql
- RESTful API

## Quick Start

### Prerequisites
- PHP 8.2 or higher
- Composer
- Node.js 18+ and npm
- Git
